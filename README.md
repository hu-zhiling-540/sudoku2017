# Sudoku Solver 2017

Formulate the Sudoku problem as a single CSP by giving the variables their domains and the constraints. 

## already had

  - Naive Backtracking
  - Backtracking + Forward Checking
  - Backtracking + Forward Checking with MRV

## To be improved

 - Backtracking + Forward Checking with MRV
